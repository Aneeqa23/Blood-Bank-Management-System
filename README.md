# Blood Bank Management System

A Java-based management system designed to streamline blood donor registration, blood inventory tracking, and request management. The project was developed to apply object-oriented programming concepts and GUI development techniques.

## Features

- Donor Registration
- Blood Inventory Management
- Blood Request Processing
- Search and Filter Records
- User-Friendly Interface
- Data Organization and Tracking

## Technologies Used

- Java
- Java Swing
- Object-Oriented Programming
- Event Handling
- GUI Development

## Learning Outcomes

This project was developed to strengthen understanding of:

- Object-Oriented Programming
- Java GUI Development
- Data Management Systems
- Event-Driven Programming
- Software Design Principles

## Future Improvements

- Database Connectivity
- User Authentication
- Blood Stock Notifications
- Hospital Integration
- Advanced Reporting Features
